# Fishynfo
Fishynfo is a website for getting info about fishes

Live Demo: [https://pari55051.github.io/fishynfo/]


## Overview

### Tech Stack
- HTML
- CSS
- Vanilla JS

### Features
- animated fish moving about in their habitat
- hover on fish to get their info
